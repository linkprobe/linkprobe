# LinkProbe

Metadata extraction, link previews, and URL-powered content experiences.

LinkProbe is an open-source toolkit for extracting, resolving, and transforming metadata from URLs.

Whether you're building a blogging platform, CMS, social feed, documentation website, or any application that relies on rich link previews, LinkProbe provides a consistent and extensible foundation for working with webpage metadata.

## Features

- Open Graph metadata extraction
- Twitter Card metadata extraction
- HTML metadata extraction
- Metadata resolution across multiple sources
- TypeScript-first API
- Framework-agnostic core
- React integrations
- Next.js integrations
- URL-to-image workflows
